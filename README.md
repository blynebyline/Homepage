# Responsive Homepage

A simple responsive homepage built as part of The Odin Project curriculum. This project focuses on practicing modern HTML and CSS layout techniques, especially Flexbox/Grid and responsive design principles.

## 📖 About the Project

This project is a responsive landing page designed to adapt to different screen sizes (desktop, tablet, and mobile). It was built to strengthen fundamental front-end skills such as layout structuring, responsiveness, and clean UI design.

## 📱 Responsive Design

The layout adapts to:

- 📱 Mobile devices
- 📟 Tablets
- 💻 Desktop screens
